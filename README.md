## Feature

- Multi-platform: `macOS` `Windows` `Linux`
- Configuration via file: `~/.wa/setting.json`
- Custom scripts: `~/.wa/scripts` is the script root path, please use a relative path, such as `./user.js`
- Custom Shortcut
- Spotlight Search
- Check for updates at startup
